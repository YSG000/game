## 🧹 파일 정리 & 배포용 만들기

**💬 "남길 html / 버릴 html 구분해줘 → SHdev 지우고 idiom을 game.html로"**
🤖 중복 `SHdev.html` 삭제, `idiom.html → game.html` 이름 변경, 라우팅·링크 전부 갱신.

**💬 "팀원한테 공유할 때 URL만? / 서버는 내 C드라이브에서 켜두면 돼? Z에서 다시 켜?"**
🤖 순위 공유는 **파일이 아니라 서버(HTTP)** 통로. **서버는 내 PC(C)에서 그대로**, Z엔 열 파일만.
**💬 "저 게임으로 들어오는 것만 방화벽 허용 가능해?"**
🤖 **파이썬 서버 + 8000 포트 + 사내망**으로만 여는 방화벽 규칙 추가(관리자 UAC).

**💬 "기억력 게임 광고·분석 기능은 삭제" / "파일로 열어도 순위 저장되게"**
🤖 광고·정밀분석 버튼 제거 + 각 게임에 **서버 주소 자동 접속**(file:// → 내 서버) 심음.
